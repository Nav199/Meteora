# Meteora - page
É uma page feita com bootstrap com o objetivo de colocar em prática o conhecimento adquirido da plataforma Alura.

Principais Características:

    Design Responsivo e Amigável:
    Explore um layout moderno e adaptável, criado utilizando as robustas classes de grid do Bootstrap.

    Barra de Navegação Intuitiva:
    Nossa barra de navegação, construída com a versátil navbar do Bootstrap, oferece uma experiência de navegação simplificada.

    Estilo Personalizado:
    A estética da página Meteora é única, combinando elementos visuais atraentes com uma paleta de cores cuidadosamente escolhida.
